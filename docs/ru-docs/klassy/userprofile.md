# UserProfile

{% hint style="success" %}
Этот раздел успешно завершен.
{% endhint %}

```python
user_id (int): Идентификатор пользователя.
name (str): Имя пользователя.
avatar_url (str): URL аватара пользователя.
caption (str): Подпись в профиле пользователя.
is_agent (bool): Указывает, является ли пользователь агентом.
default_achievement_image_url (str): URL изображения стандартного достижения.
school (dict): Информация о школе пользователя.
school_city (str): Город, в котором находится школа пользователя.
school_name (str): Название школы пользователя.
grade (str): Класс в котором учится пользователь.
level_data (dict): Информация о уровне пользователя.
level (int): Уровень пользователя.
gained_xp (int): Полученный опыт пользователем.
available_xp (int): Доступный опыт для пользователя.
progress (float): Прогресс к следующему уровню.
```
